    1. What is Semantic HTML? 

Semantic HTML is the use of meaningful tags -- meaningful to browsers in how they display content.  They help make a website more accessible to those using screen readers and other devices.

    2. What is HTML used for? 

HTML is the markup language of the internet. It provides the basic structure and content of a website/webpage. HTML stands for Hyper Text Markup Language.

    3. What is an attribute and where do we put it? 

An attribute is something that provides more information about the element it is being introduced with, and is written alongside the element tag. A good example is an image <img> element. An image must be written along with a source as <img src="filename.jpg">

    4. What is the h1 tag used for? How many times should I use it on a page?

h1 should be reserved for the most important info on the page and should only be used once.

    5. Name two tags that have required attributes

<img> and <a>

    6. What do we put in the head of our HTML document? 

Anything that's not part of the page content. Meta tags, css/script info, css styling, page title...

    7. What is an id? 

An "id" is an attribute used for special designation of an element. They can only be applied once on a page, and should not be used for stylistic purposes, but rather, reserved for functionality...

    8. What elements can I add an id to? 

Any element.

    9. How many times can I use the same id on a page? 

Only once.

    10. What is a class? 

A "class" is also an attribute, like "id", but can be applied more than once on a webpage and should be used for css styling.

    11. What elements can I add a class to? 

Any element.

    12. How many times can I use the same class on a page? 

Any number of times. 

    13. How do I get my link to open in a new tab?

target="_blank"

    14. What is the alt attribute in the image tag used for? 

Provides a short meaningful description of the image used by screen readers, and whenever an image cannot be displayed.

    15. How do I reference an id?

id is referenced by using a hashtag in front of the id name.  #idName

    16. What is the difference between a section and a div

A section is semantic html and a div can be used more generically.

    17. What is CSS used for? 

Styling HTML.

    }
