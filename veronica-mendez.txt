    1. What is Semantic HTML? 
    
        Semantic HTML is the use of HTML tags that have purposeful meaning and are interpreted as such by web browsers.
    
    2. What is HTML used for? 
    
        HTML is a markup language used to create the structure of a webpage. 

    3. What is an attribute and where do we put it? 

        An attribute is a modifier for an element, not rendered on screen, but that changes the nature of that element.  
        
        Another way of putting it is that it acts as a "special flag" that tells the browser that the element it is associated with will act according to that attribute.  The anchor element "a", for example, is modified by the attribute "href".

    4. What is the h1 tag used for? How many times should I use it on a page?
    
        h1 tag is a header tag. It is the largest header tag, and is usually reserved for the most important part of a page. Should only be used once on a page.
    
    5. Name two tags that have required attributes

        img & a

    6. What do we put in the head of our HTML document? 
    
        Anything that's not part of the page content. Meta tags, css/script info, css styling, page title

    7. What is an id? 
    
        An "id" is an attribute used for special designation of an element.
    
    8. What elements can I add an id to? 

        All elements.

    9. How many times can I use the same id on a page? 

        Only once.
    
    10. What is a class? 

        A "class" is a designation attribute, like "id", but can be applied more than once on a webpage.

    11. What elements can I add a class to? 
        
        All elements.

    12. How many times can I use the same class on a page? 

        As many times as needed.

    13. How do I get my link to open in a new tab?

        target="_blank"

    14. What is the alt attribute in the image tag used for? 

        Provides a short meaningful description of the image used by screen readers, and whenever an image cannot be displayed.

    15. How do I reference an id?

        #idName

    16. What is the difference between a section and a div

        <div> is a generic tag. <section> is more specific; it is a semantic tag that informs the browser that other elements might come with it.

    17. What is CSS used for? 

        CSS, also known as Cascading Style Sheet, is styling language used to style the content of a webpage.

        }
